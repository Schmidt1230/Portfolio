    About ChatterBox,

    ChatterBox is a Python application which was created using the GUI library customtkinter. This application allows users to create a new user and then login. Once they create a user their password is encrypted using a hash library. The application stores all data on Google's Firebase. The chat application itself allows users to add a friend based on their unique username and then send direct messages to their specified friend. The application also provides real time updates to receive new messages instantly. There is also functionallity to create a group chat and add any user, also based on their unique username.
